# ☕ Byeolstar

**별처럼 빛나는 커피 타임 — byeolstar.com**

감성적인 카페 탐방기부터 스타벅스 신메뉴 리뷰까지,  
커피 한 잔의 여유와 이야기를 사랑하는 이들을 위한 공간입니다.  
전국의 감성 카페, 트렌디한 브런치 맛집, 숨겨진 핫플레이스를 소개하며  
일상 속 특별한 순간을 함께 나눕니다.

> “별처럼 빛나는 하루, 한 잔의 커피에서 시작됩니다.”

- 데이트, 공부, 작업에 어울리는 공간 큐레이션
- 브런치 맛집 및 음료 추천 리스트
